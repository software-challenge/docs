# Setting up the (Python) development environment

Python code can be written with any text editor.
If you want additional useful features beyond a simple editor,
you should use an integrated development environment (IDE),
which really makes work easier.

Below, the setup of [Visual Studio Code](https://code.visualstudio.com/)
(an IDE from Microsoft suitable for various languages) is shown.
But environments like [PyCharm](https://www.jetbrains.com/pycharm/),
which are designed exclusively for Python, are also popular.

**Note:** Before setting up the IDE,
[Python and pip](installation-von-python) should be successfully installed.

## Setting up Visual Studio Code

### Installation

The easiest installation is done using the VS Code installer
from the [download page](https://code.visualstudio.com/Download).
After downloading the installer file,
you will be taken to a [page](https://code.visualstudio.com/docs/)
in your browser that explains the setup in great detail.

We explain the most important steps for the Software-Challenge here:

During installation, we recommend
installing all additional features.

To get Python support in VS Code, install the extension:

1.  Go to "Extensions" in the left menu
2.  Search for "Python"
3.  Select the "Python" extension by Microsoft
4.  Install the package
