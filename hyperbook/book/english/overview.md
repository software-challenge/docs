# Welcome
This is the official documentation of the programming competition
[Software-Challenge Germany](https://software-challenge.de)!

In the various sections you will find:
- Explanations of the competition
- Technical basics
- Documentation of the current game
- Explanations for developing your own computer player

### Contribute

We welcome any suggestions for improvement.  
The documentation can be edited directly on [GitHub](https://github.com/software-challenge/docs/blob/main/hyperbook/book).
Alternatively, you can also send an email to <tech@software-challenge.de> 
or a message on [Discord](https://discord.gg/jhyF7EU).
