# The First (Programming) Steps

As is well known, beginning is tough.
Therefore, a little help is given here
to make the start into development easier.

## The Right Programming Language

The easiest way is, of course,
to use one of the official player templates (Java, Python) as a basis,
but you can also write your own clients in other languages.
This involves more work,
but if you particularly enjoy using a language or see advantages,
then it can be worth it!

So if you have enough experience
and decide to take the more difficult path,
then you should take a close look at the [XML documentation](/xml),
because you have to implement the entire communication,
including parsing the XML messages.
Furthermore, you should look at
the (unofficial) [Swift computer player](https://github.com/matthesjh/sc20-swift-client) as an example.
This can also help if you don't know Swift,
since most procedural programming languages have many similarities.
Thus, it shouldn't be too difficult
to translate the Swift code into your language.
Similarly, [the Java source code](https://github.com/software-challenge/backend/tree/main/player/src)
and [the C# source code](https://github.com/niklasCarstensen/socha-client-csharp)
can also help.

It is best to discuss the use of another programming language early on with your tutor,
to ensure that the [computer player](/grundlagen/player) can also participate in the competition.
We are happy to provide a suitable runtime environment
for the programming language on the [competition system](/grundlagen/contest).

## Further Reading

Comprehensive documentation exists for most programming languages.
Some recommended ones are listed here.
The interested reader is invited
to add their own recommendations in the appropriate area.

### Java

The comprehensive (English) official documentation for Java
can be viewed [at Oracle](https://docs.oracle.com/en/java/javase/11/core/java-core-libraries1.html).


### Python

For Python, there is documentation provided directly by the Python organization.
This is available for version 3.12.8 [here](https://docs.python.org/3.12/)
and also contains a Python tutorial.
If that seems a bit confusing,
you can also check out the well-known platform [w3schools](https://www.w3schools.com/python/)
on the topic of Python.
Both resources are in English,
but many more introductions to the Python language
can be found on the internet, both in text and video format.
