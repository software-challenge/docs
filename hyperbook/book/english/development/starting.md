# Getting to know the game

In order to write a computer program that can play a game,
you must first understand the game yourself and be able to play it.
While doing so, you can already develop ideas and strategies
that you later want to implement in your [computer player](/grundlagen/player).

You can find the game rules [here](/spiele/26_piranhas/regeln.md).

The game itself can be played with two players
or alone against the [random player](/grundlagen/player#der-zufallsspieler)
using the [game server](/grundlagen/server).
It is recommended to do this several times before starting to program
in order to get a feel for the game.
